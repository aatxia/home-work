# React 

## Опис 
Цей проект створено в рамках практичного завдання з вивчення React. Мета проекту — демонстрація вміння створювати та використовувати функціональні компоненти (`Button` та `Input`), передавати їм `props` (властивості) та обробляти події користувача.
Проект ініціалізовано за допомогою **Vite** та стилізовано за допомогою **Tailwind CSS**.
